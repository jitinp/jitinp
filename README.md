## What I'm Building

**[LFG Agent](https://lfg.run)** — An autonomous software development platform that uses AI agents to ship production code. Think product manager agent → architect agent → developer agent → QA agent, all coordinated through an orchestration layer.

**[Mags.run](https://mags.run)** — Secure sandboxed execution environments for AI agents. Built on Firecracker microVMs with persistent storage via JuiceFS/S3. Every agent gets its own isolated runtime.

**[Kitereach](https://kitereach.ai)** — An automated marketing agent that runs campaigns end-to-end.

## What I Think About

I write a lot about the shift from traditional software development to agentic workflows — how background agents, sandboxed execution, and AI orchestration are reshaping what it means to build software. Most of that lives on [LinkedIn](https://www.linkedin.com/in/jitinp/).

The short version: we're moving from developers-who-use-AI-tools to AI-agents-who-use-developer-tools. The infrastructure layer (runtimes, state persistence, secure execution) is what makes that real.
